# Network-Slicing
توی شبکه‌های مدرن، آدم‌ها (یا سازمان‌ها) به‌جای کانفیگ کردن دستی شبکه، نیّت‌شون رو با زبان طبیعی می‌گن (مثلاً: «برای ویدئوکنفرانس تأخیر کم باشه»). سیستم‌های جدیدی هستن به اسم Intent-Based Networking (IBN) که این “نیّت” رو می‌فهمن و خودشون شبکه رو تنظیم می‌کنن.
